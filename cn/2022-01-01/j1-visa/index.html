J-1 签证

XiaO / 2022-01-01

此处记录一下 J-1 签证大致流程，不详述细节。

1、工作邀请函（合同）

十八般武艺齐上阵，各显神通。一言以蔽之，就是拿到对方开出的工作邀请函（合同）。这里说来话长，啥也不是的人，譬如鄙人，可不只能是 “十八般武艺” 嘛。至于更多的情况，就像张大帅说的，江湖是人情世故。如果你很江湖，那么可能并不需要十八般武艺，这听起来果然很江湖。

2、DS-2019 表

DS-2019 表是 J-1 交流访问者的身份证明文件。它是经由美国国务院授权机构，在此对国务院制定的交流访问项目的参加者进行遴选甄别后，颁发给海外申请人的约束性文件。 申请人可凭此在美国大使馆申请 J-1 交流访问签证，参加预期的交流访问项目。

  • 雇主发来申请 DS-2019 表所需的文件，填好自己所需填写的信息后，发回给雇主，由雇主申请 DS-2019 表。雇主会将纸质版本的 DS-2019 表通过 FedEx 邮寄给你。

    • DS-2019 Exchange Visitor Request Form -（全部信息自己填写）
    • J-1 English Proficiency Assessment - (部分信息自己填写，并附上可能适用的证明文件)
    • J-1 DS-2019 Host Department Request Form -（全部信息雇主填写）

  • DS-2019 表里包含的重要信息有，在填写 DS-160 表时须用到。

    • SEVIS ID: N0022222222
    • Program Sponsor Name: 雇主机构名称
    • Program Sponsor Number: P-1-20222

拿到 DS-2019 表快递后，第一页底部签字即可。

3、签证照片

申请非移民签证时，需要上传一张最近六个月内拍摄的数码照片，以便完成并提交 DS-160 表。

  • 必须为最近六个月内拍摄
  • 照片须为正方形
  • 照片最小尺寸为 600x600 像素（高 x 宽）。最大尺寸为 1200x1200 像素（高 x 宽）
  • 照片文件不超过 240kb。（不要用压缩软件对照片压缩，可能产生尾影，直接缩小照片的高宽尺寸，即可减小文件至所需大小）

4、DS-160 表

  • 网址：https://www.ustraveldocs.com/
  • 根据提示一步一步填写即可，全程相当耗时，并且在填写过程中，由于页面长时间未动，可能导致页面登出，所以在填写时，尽量将资料准备好，采用复制策略。每填写完一部分就保存一下。
  • 可将填写部分保存下来，PDF 或者 网页形式，以备往后使用。

5、I-901 表格缴费

SEVIS 是 Student and Exchange Visitor Information System （学生和交流访问学者信息系统），是为了方便移民局随时查看所有在美国际学生和学者在美国的动态，美国国土安全部的美国移民与海关执法局规定美国所有高校必须把国际学生和访问学者的信息资料输入此系统。由美国 1996 年非法移民改革和移民责任法的规定而创建的。

而 SEVIS I-901 Fee，（简称为 SEVIS Fee，因为是交给美国国土安全局的，所以中文也称国安费）是由美国国会规定，每个 I-20 表格（申请 F-1 学生签证的学生持有）和 DS-2019 表格（交流访问学者持有）的持有者都必须缴纳的（有例外）、用于维持该办公室的运转（雇佣工作人员）、并维护和完善该数据库，跟踪每个在美就读的国际学生和交流访问学者的记录以及维持他们在美的合法状态。包括他们的录取院校、录取项目、学习进度、学习状态等。

  • 网址：https://fmjfee.com/i901fee/index.html
  • 220 美元，以本地货币结算，最迟须于签证面试前 3 天付款
  • 发送付款信息到自己的邮箱，同时打印一份 PDF 文件备用

6、网上预约面签时间并支付签证申请费

  • 网址：https://cgifederal.secure.force.com/
  • 邮箱密码登录，填写相关信息，注意填写 DS-160 的确认号码
  • 支付签证申请费，160 美元
  • 发送付款信息到自己的邮箱，同时打印一份 PDF 文件备用

7、签证面试的材料准备

  • 工作邀请函或合同；
  • 签字的 DS-2019 表；
  • 一张签证照片；
  • DS-160 最后带有照片与一维条形码的表；
  • I-901 缴费表格；
  • 面试预约确认函（其中带有面试人的信息、面试时间与地点、DS-160 的确认号的条形码、DS-160 的缴费号的条形码）；
  • 护照;

可能用到的信息：

  • 个人简历 PDF
  • 雇主信息 PDF
  • 交流项目概述
  • 资金证明信息（雇主开的薪资信息、奖学金、或者个人银行信息等）

8、签证面试的结果

如果签证顺利的话，以上材料中只有护照会被收走，除了 DS160 确认页和 SEVIS FEE 收据不作为参考依据外，其他没有材料一概没有被收，签证官给你的单子上会明确写着签证即将准备好的信息。

如果要被 Check，签证官告诉你，你的 VISA 没问题，但是需要等一个月，但是他们会留下你的材料（比如 Resume、Invitation letter、Research plan 等等），然后把这些材料送到华盛顿审查，如果确定没有问题，就会给你签证。也有在审查过程中发现有问题而被拒签的，但是极少，一般都会拿到签证。

DS-2019 是入境美国后，能证明合法性的文件，一定要在自己的手上，且要保存好，因为在下了飞机准备入境的时候，DS-2019 表格是必须的材料。

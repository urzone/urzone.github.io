Github + Netlify + Yandex

XiaO / 2020-07-19

Github 中转仓库

将 Github 作为文件中转仓库站，用于为 Netlify 自动提供文件源。

Sourcetree 运输工具

将 Sourcetree 作为本地与中转仓库之间的运输工具，将本地文件推送至中转仓库。首要任务即是联通本地 Sourcetree 与远端 Github 中的中转仓库。

特点

  • 免费不开源软件
  • 界面化操作

Netlify 网站托管

特点

  • 免费网页托管；
  • 自定义域名解析；
  • 自动更新的免费 SSL/TLS 证书；
  • 无限重定向服务；

步骤

  • 注册并登录 netlify，可点击 New site from Git，将 GitHub 某个仓库中的网站内容部署到 Netlify；或者不通过 GitHub 直接将包含网站内容文件夹或压缩包拖拽上传到 Netlify。

  • 添加自定义域名；

  • 检查域名的 nameservers 设置（登录购买域名的服务商账户，将 nameservers 修改为由 Netlify 提供的主机名，可能需要最多 24 小时使修使修改生效）

  • 设置网站的 https 安全访问（如果上面的主机名已经生效，点击验证 DNS 设置 即可）

  • 默认条件下可单独使用以 Netlify 子域名形式的网址访问网站，亦可通过重定向的方式将其指向网站默认域名。可设置 urz.one 或者 以 www 开始的二级域名格式 www.urz.one 为默认域名。
https://urzone.netlify.app/	https://urz.one/

Yandex 企业邮箱

特点

  • 可创建 1000 邮箱用户；
  • 每个用户 10G 的网络硬盘容量；
  • 支持 POP3、IMAP、SMTP；

步骤

  • 注册并登录 Yandex 通行证；

  • 添加域名，并设置 CNAME，SPF 记录和 DKIM 签名；

    • CNAME 主机名 ：mail 指向： domain.mail.yandex.net
    • TXT 主机名： @ TXT值 ：v=spf1 redirect=_spf.yandex.net
    • DKIM 签名

  • 在网页中添加邮箱登录框：（下面代码中的 yourdomain.tld 替换成个人域名）

<form method="post" action="https://passport.yandex.com/for/yourdomain.tld?mode=auth"> 
 
<div class="label">Username:</div>
<input type="text" name="login" value="" tabindex="1"/>
<div class="label">Password:</div>
<input type="hidden" name="retpath" value="http://mail.yandex.ru/for/your_domain_name">
<input type="password" name="passwd" value="" maxlength="100" tabindex="2"/> <br>
 
<label for="a"><input type="checkbox" name="twoweeks" id="a" value=«no» tabindex="4"/>don't remember me</label> (<a href="https://yandex.ru/support/passport/#foreign-pc">what's this</a>)
 
<input type="submit" name="In" value="Log in" tabindex="5"/> </form>
  • 邮箱别名。每个邮箱，有一个 options 选项，其中有一个 Mailbox Alias 选项，可无限添加别名；
  • 默认邮箱 。指定默认邮箱，该邮箱可接收发送到域中的所有以不存在的用户的的邮件；
    • Go to the Settings page.
    • In the Mailbox for lost emails field, enter the name or login of the employee who lost emails are sent to. （从显示的下拉菜单中选择接收邮箱）
    • Click Save.
  • 邮箱类应用的登录设置，须网页邮箱并启用 IMAP 或 POP3；
    • Click in the upper-right corner.
    • Click All settings.
    • Click Email clients.
    • Turn on the From the imap.yandex.com server via IMAP option.
    • Save the changes.
SMTP / POP3 

POP3：pop.yandex.com 开启SSL  端口 995

SMTP：smtp.yandex.com 开启SSL 端口 465 # Outgoing mail

IMAP：imap.yandex.com 开启SSL  端口 993 # Incoming mail

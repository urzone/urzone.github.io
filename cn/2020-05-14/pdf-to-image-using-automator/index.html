用 Automator 将 PDF 批量转换为高清图片

XiaO / 2020-05-14

Mac 系统中的 Preview 软件，可以轻松查看 PDF，并将其中选中的某一页另存为图片格式，比如，JPG，PNG 等。

那么如果需要将一个很多页的 PDF 文件批量转换为图片格式呢？这个其实也是很简单的，我们可以使用 Mac 自带的 Automator 来自动化完成批量处理。

  • ⌘ + space 输入 Automator，打开 Automator；
  • ⌘ + N 新建一个应用，类型选择 Applicaiton；
  • 添加批量处理的动作：
    • 在动作列表里输入 render PDF pages as images，以找到这个动作，将其拖入右侧工作流中；
    • 第二个动作，列表查找框中输入 rename finder items，以找到这个动作，将其拖入到右侧工作流中前一个动作的后面。此时 Automator 会提示是否创建输入文件的副本，以保存我们的原始文件，点击添加即可；
    • 最后一个动作，列表查找框中查找到 Move finder items；
  • 按需求修改每个动作的行为：
    • 第一个图片渲染动作里，我们可以选择图片的格式和分辨率；
    • 第二个动作中的行为方式，选择 Make sequential，其他选项可自定义；
    • 最后一个动作，选择批量转换生成的图片的存储位置；
  • 保存应用 ⌘ + S，选择一个你喜欢的名字，比如 PDF2PNG.app，放到桌面上；

如此，该批量转换工具就完成了，你只需要把需要转换的 PDF 文件，拖拽到该应用，图片就转换完成了。

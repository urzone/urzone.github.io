MarkDown 表格与希腊字符的 LaTeX 命令

XiaO / 2021-05-29

用 Markdown 语法创建表格着实是一件令人头疼的事情，其不仅输入较麻烦，而且查看和修改数据也不够直观。Table Convert Online 是一款在线表格格式转换工具，该工具可以快速将一种语法格式的表格转换为另一种语法格式输出。

表格导入支持手动制表，URL 爬取，Excel 表格，HTML 文件等，表格导出所支持的格式更是多样，甚至可以直接导出图片。具体使用教程，请参照官网。

将网址中 $\LaTeX$ 语法的希腊字符表格转换为 MarkDown 格式，即得如下赏心悦目的表格。

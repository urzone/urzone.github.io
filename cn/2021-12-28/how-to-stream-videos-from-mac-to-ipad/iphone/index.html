从 iOS 访问 macOS 中的视频文件

XiaO / 2021-12-28

这大冬天的，其实只是想换个姿势看文 (dian) 件 (ying) 而已，譬如，躺（平）在床上。

无线网络

将 iOS 与 macOS 置于同一无线网络下，即所有设备 ip 地址的前三个字段都是相同的，譬如，除 x 外，都是 18.188.188.x 。

macOS 相关设置

  • 打开系统设置 System Preferences，点击分享 Sharing;
  • 勾选文件分享 File sharing，并在选项中选中 SMB 文件分享以及 Windows File Sharing；

iOS 相关设置

  • 通过 iOS 自带的文件 Files app 访问 macOS；
    • 读取较大文件时反应较慢；
    • 部分视频格式不支持；
  • 通过第三方软件 Documents app 访问 macOS，macOS Server 中填写 Mac 的 ip 地址，同时填写上电脑的登录用户名与密码；
    • 读取较大文件时反应足够；
    • 部分视频格式不支持；
    • 视频文件较大时，声音有卡顿延迟；
    • 支持画中画模式；
  • 通过第三方软件 VLC app 访问 macOS，macOS Server 中填写 Mac 的 ip 地址，同时填写上电脑的登录用户名与密码；
    • 只能读取视频文件；
    • 文件读取流畅；
    • 不支持画中画模式；

如此，打完收工，去超市搬运干粮。

M3U8 文件下载

XiaO / 2021-05-30

M3U8 文件是指 UTF-8 编码格式的 M3U 文件。M3U 文件是记录了一个索引的纯文本文件，打开它时播放软件并不是播放它，而是根据它的索引找到对应的音视频文件的网络地址进行在线播放。

M3U 是一种播放多媒体列表的文件格式，它的设计初衷是为了播放音频文件，比如 MP3，但是越来越多的软件现在用来播放视频文件列表，M3U 也可以指定在线流媒体音频源。很多播放器和软件都支持 M3U 文件格式。

啰嗦提示：请使用基于 Chrome 内核的浏览器进行操作，其他浏览器可能也许同样能用，但是不确定。

M3U8 下载工具

油猴插件 Media-source-extract 可自动检测并提取 M3U8 文件，所提取文件分为视频文件部分与音频文件部分。

ffmpeg 合并音视频文件

ffmpeg -i "path/to/audio.mp4" -acodec copy "audio.aac" # 将下载的 mp4 格式的音频文件转换为 aac 格式

ffmpeg -i "path/to/video.mp4" -i "path/to/audio.aac" -c copy -shortest "Final.mp4" 合并音视频文件

Convert ts files to mp4
ffmpeg -i ts_file_path -acodec copy -vcodec copy -f mp4 path+name.mp4

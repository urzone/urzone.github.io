macOS 系统降级

XiaO / 2021-01-27

人总是在欲望中想着去尝试，而又每每在尝试后陷入后悔。这似乎是人性的弱点。

三种恢复系统的快捷键

  • 通过 macOS 恢复功能启动：基于 Intel 处理器的 Mac 上，在 Mac 开机时立即按住 Command (⌘)-R，直至看到 Apple 标志或其他图像。
  • 在基于 Intel 的 Mac 上，您可以在启动时使用 Option-Command-R 来升级到与您的 Mac 兼容的最新版 macOS。
  • 在之前使用 macOS Sierra 10.12.4 或更高版本并基于 Intel 的 Mac 上，您可以在启动时使用 Shift-Option-Command-R 来安装 Mac 自带的 macOS 或与它最接近且仍在提供的版本。

三种恢复系统的途径

  • 通过 Time Machine 恢复到之前的备份的系统；[可能需先抹掉整个磁盘]
  • 重新安装 macOS，即联网状态下自动下载安装；
  • 下载所需版本的系统，并创建可引导的安装器；

无意间将系统恢复到了 2014 年的备份，桌面上居然还有当年的硕士论文稿件，恍若隔世一般呐。有时候在想，如果时光能够倒流，很多事情会不会就不一样了。

分子对接 （Molecular docking）

分子对接是一种预测两个分子彼此结合形成稳定复合物时一个分子相对于另一个分子的优选取向的方法。它可以用来预测分子结合模式以及分子结合时的亲合力。

List of protein-ligand docking software

autodock

MGL Tools, used to prepare the protein and ligand files. However, a 32-bit app is not supported on macOS 10.15.x, which means you will get “Bad CPU type in executable” in Terminal.

下载 autodock 或 autodock vina，解压至 ~/ 根目录，将其路径以别名的方式加入 shell 解释器的配置文件。

alias vina='~/autodock_vina_1_1_2_mac_catalina_64bit/bin/vina'

alias autodock='~/autodock4/autodock4'
alias autodock='~/autodock4/autogrid4'

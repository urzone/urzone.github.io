pip 工具

XiaO / 2021-02-11

pip 操作

pip 安装 python 包

pip install Somepackage1  Somepackage2 # 安装包
pip install --upgrade Somepackage1  Somepackage2 # 更新包
pip uninstall Somepackage1  Somepackage2 # 安装包

pip 对所有的安装包进行操作

alias pip-upgrade="pip freeze --user | cut -d'=' -f1 | xargs -n1 pip install -U" # 设置 pip-upgrade 别名更新所有 pip 安装的包

pip list -o --format columns|  cut -d' ' -f1|xargs -n1 pip install -U # 更新所有 pip 安装的包

pip freeze | xargs pip uninstall -y # 卸载所有 pip 安装的包

所用工具

docx2pdf

pip install docx2pdf
docx2pdf input_file.docx # 单个文件转换

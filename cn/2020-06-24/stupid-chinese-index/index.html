实时影响因子的计算

XiaO / 2020-06-24

每年期刊引证报告（Journal Citation Report, JCR）所公布的期刊影响因子 (Impact Factor, IF) ，其数据反映的是期刊在所公布年份前一年的表现情况。通常 JCR 会在每年六七月发布，而每年二月份左右大部分期刊在前一年的的引用数据已经统计完成，故而可在 JCR 正式发布之前预测一下期刊的表现。

IF 的计算公式

公式中的发表文章，即可引用项目，是指对特定研究领域的学术体系做出实质贡献的文章，包括研究 (article)，综述 (review) 以及会议论文 (proceedings paper)，不包括 editorial materials, letters, meetings abstracts, corrections 等其他形式的内容。

操作步骤

  • 进入 Web of Science 网站，数据库选择 Web of Science Core Collection，所需查询的期刊的名称如 biomaterials，类别选择 Publicaiton Name。我们在 2020 年查询的是期刊其 2019 年的表现，那么所需的是 2017 和 2018 年两年的统计数据，所以时间选择自定义，起止时间分别为 2017 和 2018 年，最后点击搜索即可。例如期刊 biomaterials 的统计数据。

  • 期刊 biomaterials 在 2017 和 2018 年共发表 1149 篇文章，在左下角的 Document Types 中提炼出 ARTICLE 和 REVIEW，剔除 9 篇修正申明，故期刊 biomaterials 在 2017 和 2018 年共发表 1140 篇文章。

  • 在右上角点击 Create Citation Report 生成引用报告，可见期刊 biomaterials 在 2017 和 2018 年发表的 1140 篇文章在 2019 年被引用 12323 次。

计算结果

于是乎：

  • biomaterials 2019: 12323/1140 = 10.810
  • journal of chromatographic science 2019: 324/250 = 1.296
  • acs nano 2019: 39543/2615 = 15.122
  • theranostics 2019: 7042/785 = 8.971
  • pharmaceutics 2019: 1559/342 = 4.558
  • nature nanotechnology 2019: 10199/318 = 32.072

通过改变上述查询时间范围，即可计算某期刊的实时影响因子。说来说去，还是这 Stupid Chinese Index。

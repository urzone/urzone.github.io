关于 Homebrew 的小故 (shì) 事 (gù)

XiaO / 2021-06-03

Homebrew 下载龟速的解决方案

brew --cache # 查看 Homebrew 所有下载文件的存放位置 
~/Library/Caches/Homebrew/downloads/ # Homebrew 下载的安装包的存放位置

安装特殊版本软件

通过 Homebrew 安装特殊版本的软件时，可（yī）能（dìng）会遇到 SHA256 mismatch 错误，进而导致无法安装。此时，可直接将 formular 中的的 SHA 无需验证即可。

Error: SHA256 mismatch
Expected: 71e11aad1e3ac68d775abe2c488e55484c74e0cd144ab028df63aaa52c611ab2
  Actual: 717cfcd11cea7c770f32aa48c5d1226cb9a9374a2749f43e72deb073e3ec286b
  
brew edit AppName # 打开 fomular

sha256 :no_check # 编辑 fomular 中的 SHA 值为不检查

清理工作

完成安装工作后，可以使用brew cleanup来清理刚刚产生的缓存文件。

brew cleanup # 清理刚刚产生的下载缓存文件

brew cleanup --prune 0 # 强制清理 n 天以前的缓存文件，包括下载文件以外的缓存

手机漫游

XiaO / 2023-02-25

我们常说的手机信号指的是手机网络运营商所建立的手机信号基站所发出的无线电信号（蜂窝移动网络）。这些信号可以被手机接收并转换为通话、短信和数据等各种信息。手机信号的强度取决于手机基站的覆盖范围、地形、建筑物等多种因素。在信号强度较弱或覆盖范围较小的地区，手机可能会遇到信号不稳定、丢失信号等问题。一个极端的例子是，当你在国外旅行，而你的手机网络运营商根本不提供漫游服务时，你的手机将无法接收到任何信号。

WiFi Calling 是一项基于 WiFi 网络的电话服务。人们可以使用该功能进行手机通话和发送短信，其操作和我们平常使用手机完全一样，唯一的区别是 WiFi Calling 使用的网络是我们的手机所连接到的任何可用 WiFi 网络。这项服务旨在帮助用户在信号较弱或不可用的地方保持通讯。

当我们身处手机网络运营商的覆盖范围之外时，如果该运营商提供漫游服务，我们的手机将连接到其他运营商的基站发出的信号。因此，如果我们需要拨打电话或发送短信，我们将使用第三方运营商的基站设备，这将导致我们自己的手机网络运营商收取高额费用。具体费用额度由我们的手机网络运营商和第三方手机网络运营商之间的协议确定，我们无法改变。但是，如果我们使用 WiFi Calling，费用将和我们没有漫游时一样。

为了启用 WiFi Calling，需要检查一些设置以确保你的手机可以使用该功能：

  • 首先，你需要确定你的手机网络运营商是否支持 WiFi Calling 功能。例如，Tello 提供 WiFi Calling 功能，但不提供手机漫游服务。如果你是 Tello 用户，可以登录 Tello 在线账户，在 “My Settings” 中找到 “Wi-Fi Calling” 选项，填写紧急 E911 地址即可注册开启该功能。

  • 其次，你需要一部支持 WiFi Calling 的手机，并在手机上启用 WiFi Calling 功能。你可以在手机的设置菜单中找到该选项：Setting - Phone - WiFi Calling - WiFi Calling on this phone。

  • 最后，你所连接的 WiFi 网络不能限制使用 WiFi Calling。这是因为一些公共 WiFi 网络（如机场、酒店等）可能限制使用某些服务，包括 WiFi Calling。如果你不能使用 WiFi Calling，可以尝试连接其他 WiFi 网络，或者联系 WiFi 管理员以了解是否有限制。

当你成功开启 WiFi Calling 后，你的手机信号显示位置将显示原来的手机运营商的名字，右侧将增加一个 WiFi 图标或显示为“WiFi Calling”。这表示你现在正在使用 Wi-Fi 网络进行通话，而不是传统的蜂窝网络。这是一个很方便的功能，可以让你在网络信号较弱或无法接收蜂窝网络信号的地方，如地下室或某些建筑物内部，依然能够进行通话。同时，通过 WiFi Calling，你也可以避免在漫游时支付高额的通讯费用。

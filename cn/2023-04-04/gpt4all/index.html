GPT4All

XiaO / 2023-04-04

GPT4All，一个使用 GPT-3.5-Turbo 生成的语料库在 LLaMa 的基础上进行训练而来的助手式的大语言模型。

  1. 从 Direct Link 或 [Torrent-Magnet] 下载 gpt4all-lora-quantized.bin 文件；
  2. 下载此仓库，并将前面下载的文件放在chat 目录下：
git clone https://github.com/nomic-ai/gpt4all.git
  1. 运行适用于您的操作系统的相应命令：
   - M1 Mac/OSX: `cd chat;./gpt4all-lora-quantized-OSX-m1`
   - Linux: `cd chat;./gpt4all-lora-quantized-linux-x86`
   - Windows (PowerShell): `cd chat;./gpt4all-lora-quantized-win64.exe`
   - Intel Mac/OSX: `cd chat;./gpt4all-lora-quantized-OSX-intel`
  1. 老爷机旧系统，如 Intel Mac/OSX Catalina 10.15.7.，在运行时会遇到如下报错：
dyld: Symbol not found: __ZNSt3__113basic_filebufIcNS_11char_traitsIcEEE4openEPKcj
  Referenced from: ~/gpt4all/chat/./gpt4all-lora-quantized-OSX-intel (which was built for Mac OS X 13.0)
  Expected in: /usr/lib/libc++.1.dylib

在老爷机旧系统 Intel Mac/OSX Catalina 10.15.7. 的终端执行如下代码：

git clone https://github.com/zanussbaum/gpt4all.cpp
cd gpt4all.cpp
make chat
cp chat ../gpt4all/chat/gpt4all-lora-quantized-OSX-intel-old

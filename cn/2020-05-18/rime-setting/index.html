更新 Rime 配置

XiaO / 2020-05-18

现在越来越觉得，好像做什么事情都需要理由。那好吧，某使用 Rime 的理由如下：

  • 个人隐私（相对现在流行的云词典输入法而言）；
  • 一劳永逸（相对它的学习成本与备份迁移而言）；
  • 关于词库，自己养一段时间就形成自己的词库了；
  • 关于习惯，这是个好东西，如果不用转换输入法；

某惯用小鹤双拼 + 系统英文输入法。

安装

brew install --cask squirrel

Rime 配置文件 说明 1 说明 2

配置文件采用 UTF-8 编码，格式为 YAML 文本。Mac 下文件夹所在地址：~/Library/Rime/，其中文件结构如下：

├── default.custom.yaml # 全局配置
├── squirrel.custom.yaml # 外观定制

├── custom_phrase.txt # 个人短语定制

├── double_pinyin_flypy.schema.yaml # <方案标识>.schema.yaml - 预设输入方案
├── double_pinyin_flypy.custom.yaml # <方案标识>.custom.yaml - 用户对预设输入方案的定制信息

# <词库名>.dict.yaml - 词库
├── luna_pinyin.dict.yaml # 主词库
├── luna_pinyin.extended.dict.yaml # 副词库 -
├── luna_pinyin.cn_en.dict.yaml # 副词库 - 中英文混合短语
├── luna_pinyin.hanyu.dict.yaml # 副词库 - 汉语大词典
├── luna_pinyin.poetry.dict.yaml # 副词库 - 唐诗宋词、千家集、楚辞、诗经
├── luna_pinyin.sgmain.dict.yaml # 副词库 - 电脑软件分类、计算机术语、全国省地市区县地名、穴位、西药药名、经典小说、音乐专业词库、网络流行新词

├── installation.yaml # 安装信息
├── user.yaml # 用户状态信息
├── build # 部署输入方案时产生文件所在的文件夹

# <词库名>.userdb
└── luna_pinyin.userdb # 记录用户用字习惯中的文件夹（自动生成）

全局配置 (default.custom.yaml)

# default.custom.yaml, 全局生效
patch:
  schema_list:
    - schema: double_pinyin_flypy # 小鹤双拼
  switcher/hotkeys:
    - F4
  switcher/save_options:
    - full_shape
    - ascii_punct
    - simplification
    # - zh_hans
    # - emoji_suggestion
  menu/page_size: 5 # 定制每页候选数
  # 更改切换中西文的快捷键
  ascii_composer:
    good_old_caps_lock: true
    switch_key:
      Shift_L: noop # 关闭 Shift_L 的中英文切换。需开启输入方案中处理西文模式及中西文切换功能 - ascii_composer
      Caps_Lock: commit_code # 开启大写字母直接上屏。需开启输入方案中处理西文模式及中西文切换功能 - ascii_composer
      Shift_R: noop
      Control_L: noop
      Control_R: noop
      Eisu_toggle: clear

扩展词库

  • 设置主词库。在 *<方案标识>.custom.yaml 中将 translator/dictionary 的值设置为主词库的名字即可：
patch:
  translator/dictionary: luna_pinyin
  • 添加副词库。luna_pinyin.dict.yaml 文件，在其中引入自己需要的副词库：
# luna_pinyin.dict.yaml
---
name: luna_pinyin
version: "2020.05.18"
sort: by_weight
use_preset_vocabulary: true
# 如下为自己的词典引入明月拼音扩展词库
import_tables:
  - luna_pinyin.extended
  - luna_pinyin.hanyu
  - luna_pinyin.poetry
  - luna_pinyin.cn_en
  - luna_pinyin.sgmain
...

  • 制作副词库

    • 深蓝词库转换 for windows
    • 繁简转换：OpenCC
brew install opencc
opencc -h # 帮助信息

Rime 输入法 使 (xí) 用 (guàn) 说 (yǎng) 明 (chéng)

  • 中英文混合输入，当需要输入英文的时候：
    • 或用 Caps Lock 键切换到大写英文状态，输入英文的第一个大写字母后，再次 Caps Lock 键取消大写锁定，继续输入英文并用空格或者回车键上屏；
    • 或用 shift + 字母按键 输入所需要英文的首字母，而后直接字母按键输入小写英文，最后空格或者回车键上屏。
  • 输入编码后，如果不需要候选字而是希望输入码对应的键盘字符直接上屏，可使用回车键（Return）
  • 删除误上屏的错词（调节词频）：选字光标（用上、下键）移到要删除的用户词组上，Shift + Fn + Delete 键删除即可；
  • Escape 键 清空未完成的输入；
  • 关闭 Rime 的中英文切换，Rime 仅用于中文输入以及临时英文输入。完全英文输入则使用系统自带英文输入法；

Rime 输入法备份

多台电脑及不同系统之间同步用户词典和用户设定的备份，可编辑用户文件夹下的 installation.yaml ，添加 sync_dir: '' 语句，并补充上相应的同步位置即可：

sync_dir: '/Users/xiao/RimeSync/'
  • installation.yaml 文件中，每套 Rime 会随机生成一个 UUID 作为标识。不同 installation ID 可区分来自不同机器/系统的用户词典。可用小写字母、数字、横线和下划线自定义该 ID：
installation_id: 'Rime-setting-mac-xiao'

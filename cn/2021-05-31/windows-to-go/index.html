Windows To Go

XiaO / 2021-05-31

WTG 安装

软件准备

  1. WTG 镜像写入工具

WTG 辅助工具，用于将 Windows 镜像写入到 U 盘、移动硬盘等便携设备上，并且写入移动设备上的 Windows 系统能随处在不同的电脑硬件上直接运行，包括基于 Intel 的 Mac 硬件。该软件纯绿色免安装，即开即食。

  1. Windows 镜像

    • 微软官方镜像下载；
    • itellyou 镜像下载；
    • Windsys Project —— 据说最适合年轻人（魔改）；
    • Windows 10 LTSC —— 吻妻版（魔改）；

  2. Windows 驱动 [Mac 需求]

在 macOS 中启动「启动转换助理」，点击菜单栏中的「操作」，然后选择「下载 Windows 支持软件」

Boot Camp Assistant —> Action —> Download Windows Support Software

硬件准备

  1. WTG 移动设备

微软官方对 WTG 的要求为：接口为 USB 2.0 及以上，容量为 32 G 及以上的移动设备。可以考虑如下：

  • 微软认证移动盘
  • 合适的第三方 U 盘
  • 小容量固态硬盘
  • 高性能机械硬盘
  1. 容量 8G U 盘

该 U 盘是为装载前面下载的 WindowsSupport，以备为 Mac 硬件安装 Windows 驱动。

安装过程

在 Windows 设备上，插入移动盘 —> 打开 WTG 辅助工具 —> 在辅助工具中选择移动盘以及下载的 Windows 镜像格式文件（.WIM）—> 其他默认，点击「部署」。

WTG 使用

在 Mac 硬件下启用

  1. Mac 关机，接上 WTG 设备；开机时按住 option 键，待显示启动选项之后选择橙色的「EFI Boot」，这个「EFI Bo ot」就是先前制作的 WTG 启动盘；
  2. 插入装载有「Windows 支持软件」的 U 盘，打开 WindowsSupport 文件夹下的 BootCamp.exe 文件安装驱动。安装完成之后，自带的键盘、触控板、蓝牙、Wi-Fi 等功能就能正常使用了；
  3. 将 Windows 的滚动方向更改成和 Mac 一致的 “自然” 滚动；

在 PC 硬件下启用

进入启动引导界面，选择你的 WTG 启动盘并回车即可。Windows 自带绝大多数 PC 上硬件的驱动，所以不必像 Mac 一样考虑驱动问题。

shell 中的特殊符号

XiaO / 2020-06-12

# comments

shell 中以 # 开始的行会被 shell 忽略而不执行，可用作注释。

# This line is a comment.

| pipe

将前一个命令的输出结果传递给下一个命令，作为下一个命令的输入参数。

command 1 | command 2 

ls -s | sort -nr # -s 是 file size，-n 是 numeric-sort，-r 是 reverse，即 列出当前目录中文档及其大小，并把结果传递给 sort 命令，sort 命令按数字递减的顺序将结果在 shell 中列出

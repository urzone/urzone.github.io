物态

XiaO / 2021-10-01

物质的状态（物态）指物质出现的相。物质按保持物质理化性质的最小单位（简称粒子，包括離子、原子或分子）间的相互作用的关系分为固态、液态、气态以及电浆态。

  • 固态：粒子之间因为相互的吸引力导致粒子只会在固定的位置震动。
  • 液体：粒子之间距离仍然比较近，粒子之间仍有一定的吸引力，因此只能在有限的范围内活动。
  • 气态，粒子之间的距离较远，粒子之间的吸引力并不显著，可以随意活动。
  • 电浆态，是在高温之下出现的高度离子化的气体。

固体可分为：晶体、非晶体和准晶体。其中，晶体按不同的分类标准又可分为不同的晶体，如单晶与多晶、共晶与盐等。

单晶：整个晶体内原子都按周期性规则排列；
多晶：许多晶粒组成的固体称为多晶；

而关于共晶与盐的区分，主要是以晶体内化合物之间的相互作用类型为区分。

  • 盐：晶体内化合物之间存在大量的离子化的相互作用；
  • 共晶：晶体内化合物之间存以非离子方式相互作用；
    • 共晶中，API 与共晶形成物都存在于晶胞中，且 API 与共晶形成物之间以非离子方式相互作用；
      • 当 API 与共晶形成物为酸碱性化合物，其 $pKa$ 的差值 $\Delta pKa = pKa_{\ conjugate\ acid\ of\ base} - pKa_{\ acid} > 1$ 时，API 与共晶形成物之间将有大量的质子转移，进而导致 API 与共晶形成物的离子化以及盐的形成；
      • 当 API 与共晶形成物为酸碱性化合物，其 $pKa$ 的差值 $\Delta pKa = pKa_{\ conjugate\ acid\ of\ base} - pKa_{\ acid} < 1$ 时，API 与共晶形成物之间质子转移相当有限，API 与共晶形成物则形成共晶；
      • 此外，除上述 $pKa$ 差值的方式，可使用光谱工具等其它方式确定 API 与共晶形成物的关系。
    • 对于药物共晶而言，还须确保 API 与共晶形成物在到达药理活性位点之前解离，以便 API 与相应的药理活性位点发生作用。

共晶形成物 (coformer)：一种在晶格中与 API 以非离子方式相互作用的组分，它不是溶剂（包括水），并且通常是非挥发性的。

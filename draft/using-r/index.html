Using R

2020-04-20

Install Xcode command line tools for using commands like make, GCC, clang, perl, svn, git, size, strip, strings, libtool, cpp…

xcode-select --install

Install Homebrew

- /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

brew list --versions
brew outdated
brew update
brew cleanup

Install Homebrew Cask brew install caskroom/cask/brew-cask

ursing brew cask to install and update R and Rstudio:

brew cask install r
brew link --overwrite r

brew cask uninstall --force
brew cask upgrade

brew update && brew upgrade && brew cleanup

During upgrading, If you get an error of the type Error:

Cask 'hex-fiend-beta' definition is invalid: invalid 'depends_on macos' value: ":lion", where hex-fiend-beta can be any cask name, and :lion any macOS release name, run the following command:

/usr/bin/find "$(brew --prefix)/Caskroom/"*'/.metadata' -type f -name '*.rb' -print0 | /usr/bin/xargs -0 /usr/bin/perl -i -pe 's/depends_on macos: \[.*?\]//gsm;s/depends_on macos: .*//g'
